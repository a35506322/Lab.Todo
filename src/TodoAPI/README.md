# Lab.TodoAPI

## Tech Stack
- ASP.NET Core 10
- Minimal API

## Features
- [] Minimal API + 垂直切割
- [] Swagger 文件
- [] JWT 驗證
- [] Model 驗證 + 統一回傳驗證錯誤訊息
- [] 資料存取工具 (Dapper+ Entity Framework Core)
- [] 測試專案 (MSTest)
- [] Logging (Serilog)
- [] Exception Handling (Middleware)
- [] HttpClient 最佳示範
- [] CORS 設定
