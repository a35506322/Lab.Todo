# Lab.TodoWeb

## Framework

- [x] Vue 3
- [x] Vite
- [x] TailwindCSS
- [x] PrimeVue UI

## Features

- [x] 使用 Sakai 官方 Dashboard Template
- [x] PrimeVue UI 組件
- [x] TailwindCSS 樣式
- [x] Axios 串接 API
- [ ] Axios 攔截器
- [ ] PrimeVue Form 驗證
