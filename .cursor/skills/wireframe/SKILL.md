# 繪製文字線稿圖

## 使用時機

- 使用者提供截圖，要轉成文字線稿圖。
- 使用者描述 UI 需求，要先產出低擬真版面草圖。

## 工作步驟

1. 先閱讀使用者截圖或文字描述。
2. 抽出主要區塊（頁首、工具列、內容區、表格、分頁等）。
3. 使用純文字框線輸出線稿圖。
4. 保留結構重點，不必還原所有細節。

## 輸出格式

```text
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                                                                                         │
│  Manage Products                                              [Search...        🔍]     │
│                                                                                         │
│  [+ New]  [Delete]                                                            [Export]  │
│                                                                                         │
│  ┌───────────────────────────────────────────────────────────────────────────────────┐  │
│  │ Actions  □   Code ↑↓    Name ↑↓      Image    Price ↑↓   Category ↑↓  Status ↑↓ │  │
│  ├───────────────────────────────────────────────────────────────────────────────────┤  │
│  │ ... 略 ...                                                                         │  │
│  └───────────────────────────────────────────────────────────────────────────────────┘  │
│                                                                                         │
│  [<<] [<]  [1] [2] [3]  [>] [>>]              Showing 1 to 10 of 30 products  [10 ▼]  │
│                                                                                         │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```
