# 產生資料夾結構

## 功能描述

按照使用者指定的檔案資料夾，寫出資料夾結構

## 使用方式

1. 閱讀資料夾結構
2. 資料夾結構範例
    - 在 Folder / 檔案旁使用 `# 註解說明`
    - 如太多檔案可省略僅顯示部分檔案

```text
Modules # 模組
├── Auth # 權限模組
│   ├── GetByString #取得權限
│   │   ├── Endpoint.cs # Endpoint 檔案
│   │   ├── Models.cs   # Models 檔案
│   │   └── Examples.cs # Examples 檔案
│   └── UserController
```
